__________________________________
___________Как работать___________
__________________________________

Файлы line.in, Main.java, LineFilter.bat должны находится в одной директории.
Файл line.in является входным файлом.
Входной файл содержит в себе три строки:
"имя файла с изображением для редактирования"
применяемый фильтр
"имя файла с полученным изображением"

Пример:
-----------------
inputImage.jpg
shapes
outputImage.jpg
-----------------

Для запуска нужно запустить двойным щелчком файл LineFilter.bat
